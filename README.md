# HandwrittenDigitRecognizer

![HandwrittenDigitRecognizer](https://github.com/altescy/HandwrittenDigitRecognizer/blob/master/screenshot.png)

## Requirement
- Python 2.7.11
- NumPy
- PyQt4
- Chainer 1.12.0
- scikit-learn
